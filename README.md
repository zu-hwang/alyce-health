# Alyce Health
기사 검색 애플리케이션

## 구현기능

### 기본 홈 화면 
- 최신뉴스(한국기준) 출력 적용
- 페이징 : 무한스크롤 적용

### 로그인
| no   |      ID      |  password |
|----------|:-------------|:------|
| 1 |  alyce | alyce123 |
| 2 |    hwang   |  hwang123 |

2개의 계정 사용가능

### 검색
- 검색어 입력 & 결과 출력
- 페이징 : 무한스크롤 적용
- 기사 제목 클릭 → 새 탭으로 페이지 열기

### 북마크 & 북마크 페이지
- 로그인 했을 경우 북마크 아이콘 활성화
- 홈 화면 또는 검색한 뉴스 목록 북마크 주가, 제거
- 로그인 유저의 북마크 목록 출력
- 로컬스토리지에 유저정보 구분하여 북마크목록 저장 & 불러오기


## 추가해야할 기능
- 북마크 한 기사 수정 & 저장
